2. Leads Management

2.1 What is a Lead?

A Lead is a potential customer from various channels: trade shows, website inquiries, referrals, cold calls.

2.2 Add New Lead

1 Click Leads in top nav

2 Click + New Lead

3 Fill info and Save

2.3 Bulk Import

1 Click Leads then Import

2 Download template, fill, upload

3 Click Import

Notes: Filename in English, date YYYY-MM-DD, keep headers

2.7 Import Validation Rules

Field Rule
Name Required
Leads Status Qualified, Waiting for Response, Unqualified, Waiting to be Contacted
Source Website, Referral, Email Campaign, Social Media, Event, Trade Show, Advertisement, Partner, Direct Inquiry
Date Added Multiple date formats auto-recognized
Email Valid format OR blank

2.4 Edit Lead

  • Method 1: Click Edit button
  • Method 2: Click any text field to edit inline

2.5 Fields Filter

3. Pipeline Management

3.1 What is Pipeline?

Pipeline represents active sales opportunities. TCV and quarterly revenue are calculated based on Pipeline data.

3.2 Lead to Pipeline

  1. Find a Qualified lead
  2. Change Status to Qualified
  3. Click Save
  4. Lead auto-links to Pipeline

3.3 Create Pipeline

1 Go to Pipeline page

2 Click + New Pipeline

3 Fill info and Save

3.4 Core Fields

Field Description
TCV ($) MRCx12xYears + OTC
MRC ($) Monthly Recurring Revenue
OTC ($) One-time Charge
Est. Sign Date Signing date
Est. Act. Date Activation. Empty/TBD = no forecast
Support Support team (multi-select)
Contract Term Years (1-5, default 3)

3.5 Edit Pipeline

Click Edit button to modify MRC, OTC, dates, stage, etc.

3.6 Sales Stages

Stage Description In Revenue
1) Prospecting Lead identified / Pre-qualification Yes
2) Lead Qualified Move to downstream, opportunities for BIT
3) Demo/Meeting Meeting / solution
4) Proposal Proposing solution + pricing
5) Negotiation Negotiating
6a) Deal Won Won
6b) Deal Lost Lost No
7) Activated Activated Yes
Rule: All stages count EXCEPT Deal Lost.

3.7 Pipeline List

View and manage all Pipeline opportunities.

4. Follow-up

4.1 What is Follow-up?

Follow-up records each customer communication including notes, stuckpoints, and next steps.

4.2 Add Follow-up

1 Open Pipeline detail

2 Click Follow button

3 View history (read-only)

4 Fill: Note(required), Stuckpoint(opt), Stage(opt), Next Steps/To-do(opt)

5 If adding To-do, fill Due Date

6 Click Save

Sync: Follow-up saved. To-do = new Task. Stage updates Pipeline.
